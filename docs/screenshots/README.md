# docs/screenshots/

Committed screenshot PNGs for the JAIL README and the website handoff copy.

## Rules

- **Synthetic content only.** Every committed image shows fictional **Jordan Lee** demo data (see [`../../examples/jordan-lee-demo/`](../../examples/jordan-lee-demo/)). Never commit a screenshot containing real candidate data, real companies, real URLs, or anyone's real job search.
- Screenshots of **runtime folders** (`__READY TO REVIEW/`, `05-SUBMITTED-APPLICATIONS/`, the `00-INTAKE` folders, etc.) are allowed **only** when those folders are populated with synthetic Jordan Lee data. The runtime folders themselves stay gitignored; only the PNG (synthetic content) is committed here.
- **Never** commit images from the real-URL smoke test.

## Naming

Use the `[shot: …]` labels referenced by [`../../README.md`](../../README.md) and [`../jail-public-page-copy.md`](../jail-public-page-copy.md) (e.g. `ranking-xlsx.png`, `intake-review.png`, `folder-tree.png`). The capture checklist + per-shot staging lives in [`../../examples/jordan-lee-demo/screenshots-notes.md`](../../examples/jordan-lee-demo/screenshots-notes.md).
