# Writing Links Key

A reference index of your public work — published writing, portfolio pieces, side projects, talks —
built so each piece can be **linked strategically from a cover letter** (and resume). This file is
the writer's ONLY source of link URLs: if a URL isn't here, it doesn't go in a letter.

Why this exists: inline links are quiet proof. A letter that says "I think a lot about why people
do or don't [come back to products that are supposed to help](url)" carries evidence a resume
can't — but only if the link is real, current, and matched to what the role actually values.
If you've been between roles, links are also how the letter shows the gap wasn't empty.

For every piece record: title (linked), date, a short summary, and — most important — a note on
**when it's strategic to link it**.

---

## How to choose links (multi-dimensional, not title-matching)

When picking links for a letter, weigh each piece on more than its title or topic:
1. **Substantive fit** — does the actual argument show thinking the role values?
2. **Signal value** — what the title communicates at a glance / on a recruiter's quick scan.
3. **Social proof** — visible engagement can raise an otherwise lighter piece's value, but read it
   as an input, not a quality ranking (engagement skews by age and topic).
4. **Gateway value** — an index/hub piece can be worth linking because it leads to deeper work.
5. **Portfolio balance** — the set together should show distinct strengths, not one narrow theme
   (unless the role strongly warrants it).
6. **Risk of over-narrowing** — a niche title may be perfect for an aligned role but too narrow
   for a general one; judge the framing, not just the content.
