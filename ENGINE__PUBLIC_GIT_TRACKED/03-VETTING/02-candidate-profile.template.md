**How this file is used — two kinds of content, kept apart:**
- **EVIDENCE** (Career Identity, Throughlines, Strengths, Adjacent Experience, Gaps, Misclassifications, Seniority, Career History) = what you can *demonstrate*. Used by **Profile Fit ("How They May See Your Profile")** and the Generalist-Fit part of **Company Style**.
- **PREFERENCES** (Priority Lanes, Practical Constraints, and any company/role/manager preferences) = what you *want*. Used by **Want-it (Desire)**, **Company Style**, **Practicality**, and lane assignment.
- **The wall:** a preference is never evidence that an employer will see you as qualified. Wanting a lane/mission/company must never move Profile Fit.

**This is a canonical summary derived from richer materials** (your experience bank + master profile). **Two kinds of silence, treated differently:** if the *posting* is vague or the requirement is an undocumented baseline ("strong communication"), score neutrally — silence there is not failure. But if a posting *clearly asks* for a specialization and this profile has no named evidence of it, that is **not** license for optimism — the scorer grades what a hiring team could actually see. Items in Confirmed Recurring Gaps and Boundaries are established, recurring gaps — but that list is not exhaustive; a per-job `candidate-silent` finding on an unlisted specialization is a separate, job-scoped judgment, not proof the list is complete. Material open questions get logged for you to answer.
