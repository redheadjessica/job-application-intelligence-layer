## Purpose
This file holds reusable bio language for cover letters, recruiter outreach, networking messages,
application free-text fields, speaking/intro blurbs, and profile summaries.

Use it to:
- speed up first drafts
- keep your positioning consistent across contexts
- provide multiple lengths for different amounts of space
- keep reusable bio copy separate from role-specific resume content

## Core Rules
<!-- intake: Edit these to taste. They set the tone for everything generated from this file. -->
- Keep claims grounded and interview-defensible.
- Prioritize clarity over polish.
- Tailor for mission, audience, and seniority.
- Avoid sounding generic, corporate, overly lofty, or over-rehearsed.
- Keep detailed proof in the Experience Bank (`04-experience-bank.md`), not here.
- Treat this as reusable narrative language, not a factual master record.
- {{STYLE RULE — any personal writing constraint, e.g. "no em dashes," "no exclamation points"}}

## How this file is used (conditional)
<!-- intake: Keep this note. It documents why the file exists and when it gets pulled in. -->
This file is **only** consulted during tailoring when a job is flagged `[USE BIO]` (for example, when
a posting explicitly asks for a cover letter or a written intro). In the default flow, the pipeline
produces a tailored resume and ignores this file. So fill it in only if you expect to want bio prose;
otherwise it can stay blank.
